# Data Science Capstone Project

First capstone project from the Udemy "Python for Data Science" course, focused on data manipulation and visualization.

## Overview

Hands-on project working with Pandas and NumPy for data frame manipulation, combined with Matplotlib and Seaborn for comprehensive data visualization.

## Skills Demonstrated

- Data loading and cleaning with Pandas
- DataFrame manipulation and aggregation
- Statistical analysis with NumPy
- Data visualization with Matplotlib and Seaborn
- Exploratory data analysis (EDA)

## Tech Stack

- **Language:** Python (Jupyter Notebook)
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn

## Getting Started

```bash
pip install pandas numpy matplotlib seaborn jupyter
jupyter notebook
```
